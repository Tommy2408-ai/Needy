# 🤝 Needy
**Piattaforma Open Source per il Co-Housing e il Welfare di Comunità.**

## 🌍 il Progetto (The Project)
Needy nasce per risolvere un problema semplice ma fondamentale: connettere chi ha bisogno di aiuto quotidiano con chi ha tempo da offrire all'interno dello stesso quartiere. Non è un social network, ma uno strumento di **mutuo soccorso locale**.

L'obiettivo è digitalizzare la "Banca del Tempo" rendendola accessibile ad anziani, famiglie e studenti, garantendo privacy e sicurezza.

## 🚀 Funzionalità Principali (Features)
- **Richieste Geocalizzate:** Vedi chi ha bisogno di aiuto nel raggio di 1km.
- **Sistema di Reputazione:** Guadagna "Ore" aiutando gli altri.
- **Privacy First:** I dati sensibili (documenti) sono validati ma mai esposti pubblicamente.
- **Open Source:** Progettato per essere ospitato autonomamente da Comuni o Associaizoni.

## 🛠️ Tecnologie (Tech Stack)
- **Backend:** PocketBase (Go) - Per database real-time e autenticazione.
- **Frontend:** .NET MAUI / C# - Per un'applicaizone cross-platform (Android/Windows).
- **Mappe:** OpenStreetMap.
